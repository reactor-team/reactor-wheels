# Reactor Wheels

Pre-built Python wheels for the Reactor platform.

These are GPU-specific packages that require compilation with specific CUDA compute capabilities.

## Available Wheels

| Package | Version | GPU | Python | CUDA |
|---------|---------|-----|--------|------|
| SageAttention 3 | 1.0.0 | Blackwell/B200 | 3.10 | 12.8 |

## Usage

```bash
pip install https://github.com/reactor-team/reactor-wheels/releases/download/sageattn3-1.0.0-b200/sageattn3-1.0.0-cp310-cp310-linux_x86_64.whl
```
